# DCloud REST API

Приложението представлява базов Spring Boot REST API, който демонстрира работа с пет таблици (фирми, доставчици, документи, артикули и редове в документи) чрез in-memory данни.

## Стартиране

```bash
mvn spring-boot:run
```

Приложението стартира на `http://localhost:8080`.

## Основни ендпойнти

| Метод | URL | Описание |
| --- | --- | --- |
| `POST` | `/api/login` | Проверява потребител и парола. Тяло: `{ "username": "tech", "password": "secret" }`. |
| `GET` | `/api/documents?startDate=2024-05-01&endDate=2024-05-20` | Връща всички документи за периода. |
| `GET` | `/api/documents/{id}` | Детайли за конкретен документ – код, име, баркод, количество и цена за всеки ред. |

Всички дати използват ISO формат `yyyy-MM-dd`.
